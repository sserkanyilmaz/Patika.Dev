# Insertion Sort Projesi
## Proje 1
### insertion sort -> [22,27,16,2,18,6]
#### Yukarıda verilen dizinin sort türüne göre aşamalar.
- [22,27,16,2,18,6]
- [2,27,16,22,18,6]
- [2,6,16,22,18,27]
- [2,6,16,18,22,27]
#### Big-O gösterimi
- O((n+n^2)/2)
- O(n^2)
### Time Complexity
#### Average case
O(n^2)
#### Worst case
O(n^2)
#### Best case
O(n)
### Dizi sıralandıktan sonra 18 sayısının gireceği case kapsamı
Average case
